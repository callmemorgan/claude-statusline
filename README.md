# TUI experiment screenshots

Image-only branch hosting walkthrough screenshots for the 9 experimental `configure`-TUI
interaction-model PRs (#7–#15). Not code — just assets referenced from those PRs' comments.

Captured headlessly from each branch's final commit: the mode runs under `tmux` at 142×40
in a throwaway `HOME`, keys are sent, the pane is captured (ANSI), and rendered to PNG by
`freeze`. Backgrounds, truncation, and box layout are faithful to what a user sees.

```
shots/<variant>/NN-<scene>.png   individual scenes
shots/<variant>/_walkthrough.png 2-column montage of all that variant's scenes
```
